# Cyber-Agent

Persistent autonomous engineering runtime using ChatGPT Web + Playwright.

## Features

- Persistent hidden browser runtime
- Project workspace isolation
- Autonomous Expo project scaffolding
- Validation pipeline
- Backup and rollback system
- Runtime memory system
- Execution-first orchestration

## Architecture

```txt
/runtime
/browser
/projects
/validation
/memory
```

## Usage

```bash
npm install
node index.js
```
